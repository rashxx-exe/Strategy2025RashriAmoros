# Strategy 2025 Rashri Amorós

Mi strategy se basa en un combate por turnos en el que tus fichas son todas variantes de Jeff the Land Shark (Marvel).
Cada Jeff tiene su habilidad, como:
* Basico (Es basico).
* Delfín (Ataca desde donde sea siempre que este en la misma columna (Y))
* Calabaza (Defensa, absorve los ataques del Jeff que tiene detrás)
* Venom (Tiene mucho ataque)
* Playero (Cura)
* Bezos (El Jeff Bezos augmenta en x2 el atque de los Jeffs que tiene alrededor (esta un poco bugg))

### Controles

El strategy usa puramente el dragg a excepción de:
* A/Esc para abrir el menú.
